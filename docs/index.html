Get Started in 5 Minutes

Step 1

Install

git clone https://github.com/jmenichole/Portfolio-Necromancer.git
cd Portfolio-Necromancer
pip install -e .
Step 2

Configure

portfolio-necromancer --init
# Edit config.yaml with your info

Add your API keys for AI features and data sources (optional for demo)

Step 3

Generate

portfolio-necromancer

Your portfolio will be generated in generated_portfolios/
